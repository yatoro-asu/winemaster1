# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## О проекте

Это учебный проект на Python + Jinja2, который показывает данные о вине из Excel-файла и рендерит их в HTML.

## Требования

- Python 3.11+ или 3.14
- `Jinja2`
- `openpyxl`
- `requests`

Установите зависимости из `requirements.txt`:

```powershell
python -m pip install -r requirements.txt
```

## Запуск

1. Активируйте виртуальное окружение:

```powershell
& .venv\Scripts\Activate.ps1
```

2. Запустите сервер с файлом данных из репозитория:

```powershell
python main.py
```

3. Если нужно указать свой файл данных, используйте аргумент или переменную окружения:

```powershell
python main.py --data-file data/wine3.xlsx
$env:WINE_DATA_FILE='data/wine3.xlsx'; python main.py
```

4. Откройте браузер и перейдите на:

```text
http://127.0.0.1:8000/
```

## Что делает проект

- загружает данные из Excel-файла
- группирует продукцию по категориям
- рендерит сайт через Jinja2
- показывает метки и карточки товаров

## Структура проекта

- `main.py` — HTTP-сервер и рендеринг шаблона
- `read_wines.py` — загрузка и парсинг Excel
- `template.html` — Jinja2-шаблон страницы
- `requirements.txt` — зависимости

## Примечания

Проект работает с файлом `data/wine3.xlsx` из репозитория.
Если нужно использовать другой файл, укажите его через аргумент `--data-file` или переменную окружения `WINE_DATA_FILE`.
